7. 三个竞对的实际策略

Bohrium：商业化增长型

核心特征：

  • 流量规模最大。
  • Paid Search、Paid Social、Display 都有明显占比。
  • 投放关键词围绕 google scholar、谷歌学术、consensus、elicit、arxiv。
  • 目标不是只抢 “AI for Science”，是抢科研人员每天会用的学术检索入口。

BioFord 可参考：

  • 用 Search Ads 测试文献检索和学术搜索相关词。
  • 用 YouTube 或 LinkedIn 做科研场景演示内容。
  • 但不建议一开始复制大规模 paid social/display，因为 BioFord 当前页面资产和品牌承接还仍可补足。

Edison Scientific：品牌传播型

核心特征：

  • Direct 占比 55.69%，说明用户主动访问强。
  • Paid Search 只有 3.26%，主要围绕品牌和产品词。
  • Social 以 LinkedIn 和 X 为主。
  • Referral 与 FutureHouse、媒体、VC、招聘渠道有关。

BioFord 可参考：

  • 用产品发布文章抢 AI scientist、AI for Science、scientific discovery 等概念词。
  • 用 LinkedIn 和 X 建立创始人、产品、科研场景传播。
  • 付费搜索先做品牌词、竞品词和高意图品类词，不需要一开始大预算铺开。

Biomni：学术权威型

核心特征：

  • Organic Search 占比 48.14%，三者最高。
  • Paid Search 几乎没有。
  • Referral 主要来自 GitHub 和学术网站。
  • 流量规模不大，但精准度和信任感强。

BioFord 可参考：

  • 做公开的 Model Library、Benchmark、Demo、Example Workflow。
  • 在 GitHub、bioRxiv、论文、技术博客、学术社区中建立可引用资产。
  • 把“模型能力”和“科研任务”写成搜索引擎可理解的页面。

8. BioFord 增长策略建议

第一阶段：补可见度

目标：让 BioFord 从 Similarweb 和搜索工具里“有数据可见”。

要做：

  • 建立首页 SEO 基础。
  • 建立四大智能体独立页面。
  • 建立 Model Library 公开页。
  • 配好 GA4、Google Search Console、Google Ads conversion、LinkedIn Insight Tag。

第二阶段：用小预算 SEM 验证需求

优先投放：

词组目的
ai for science、scientific ai、ai co-scientist验证品类需求
biomedical ai agent、bioinformatics ai验证 BioFord 核心定位
literature synthesis、scientific literature验证 BioScholar
molecular docking ai、protein structure prediction验证 BioModeler
protocol generator、reagent calculator验证 BioDesigner
consensus、elicit、google scholar alternative测试文献检索入口心智

不建议一开始大规模投：

  • Display
  • Paid Social 大预算
  • 泛科学百科词
  • 过宽的 AI、biology、science 类大词

第三阶段：建立长期自然流量

SEO 重点：

  • Edison 式品类页：AI for Science、AI scientist、scientific discovery。
  • Biomni 式任务页：bioinformatics AI、single-cell analysis、molecular modeling、CRISPR guide design。
  • Bohrium 式内容库放到后期做，先不要作为第一优先级。

GEO 重点：

  • 每个页面回答清楚：BioFord 是什么、适合谁、能完成什么任务、和传统工具有什么不同。
  • 增加结构化 FAQ、对比页、案例页、术语解释页。
  • 让 ChatGPT、Perplexity、Google AI Overviews 更容易引用 BioFord。

SEM 重点：

  • Google Search 先行，LinkedIn 做再营销和科研人群触达。
  • 每组广告只对应一个明确页面，不把所有流量都导向首页。
  • 先看转化质量，再扩大预算。

9. 策略摘要

BioFord 不应该只把 SEM 当成买流量，而应该用 SEM 验证关键词需求，用 SEO 承接长期搜索流量，用 GEO 建立 AI 搜索可见度；竞对里 Bohrium 代表商业投放，Edison 代表品牌传播，Biomni 代表学术权威，BioFord 最适合先组合 Edison 和 Biomni 的打法，再小规模测试 Bohrium 式投放。

14. 补充判断

竞对不是同一种增长模型。Bohrium 的核心是“内容库 + 投放”，Edison 的核心是“品牌叙事 + 产品发布”，Biomni 的核心是“学术权威 + 首页集中承接”。BioFord 当前应该先补齐 Edison 和 Biomni 这两类资产：品类页、智能体页面、模型库、Demo、Benchmark，再用小预算 SEM 去测试 Bohrium 已验证过的学术搜索入口词。

19. 推荐的 PR 与 GEO 节奏

阶段要发布的内容目标
第 1 阶段BioFord Research / Evidence Hub、四大智能体页面、Model Library 页面先让官网有可被引用的事实来源
第 2 阶段技术白皮书或预印本、GitHub demo、SDK、示例数据集建立学术和开发者可信度
第 3 阶段公开 benchmark、复现实验案例、与公开数据库结合的案例证明产品能力，不只讲概念
第 4 阶段合作机构页面、实验室试用页、高校/医院培训页获取高权威外链和真实使用证据
第 5 阶段融资/产品发布/合作 PR，LinkedIn/X 二次传播放大已有证据，不做空泛宣传

20. 外链结论

竞对的权威性不是靠“多发文章”获得的，是靠可验证资产叠加出来的。Bohrium 靠高校图书馆、论文库、开源 SDK 和国家级媒体；Edison 靠融资 PR、NVIDIA case study、投资机构页面和 benchmark；Biomni 靠 Stanford、bioRxiv、GitHub、PyPI、Hugging Face 和数据集。BioFord 应该优先做 Evidence Hub、Model Library、GitHub/SDK、benchmark 和合作机构页面，让 SEO、GEO 和 PR 都围绕同一套可信证据展开。
